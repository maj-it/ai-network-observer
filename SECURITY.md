# Politique de Sécurité

## Code Sensible

Ce repository contient du code de **démonstration éducative**.

### Exclusions Git

- `poc/keylogger/` : Non partagé publiquement
- `.private/` : Dossier local uniquement
- Logs et credentials : `.gitignore`

### Branches

- `main` : Code public (détection uniquement)
- `private-poc` : POC sensibles (local, jamais push)

### Signalement

Vulnérabilités → LinkedIn ou GitHub Issues
